# Fast React Pizza

A lightweight, fast React app for browsing a pizza menu, building an order, and simulating checkout. Designed as a starter/demo project for a single-page pizza storefront with responsive UI and simple state management.

## Key features
- Browse pizza menu with images, descriptions, and prices
- Add/remove pizzas to cart and edit quantities
- Cart summary with subtotal and simple order flow (mock checkout)
- Responsive layout for desktop and mobile
- Ready to connect to a REST API or use local/mock data

## Tech stack
- React + Vite (fast dev server)
- CSS (Tailwind)
- Node.js for dev/build tasks
- React Router, Redux

## Quick start
1. Clone repository
    ```
    git clone <repo-url>
    cd fast-react-pizza
    ```
2. Install dependencies
    ```
    npm install
    # or
    yarn
    ```
3. Run dev server
    ```
    npm run dev
    # or
    yarn dev
    ```
4. Build for production
    ```
    npm run build
    # serve preview
    npm run preview
    ```
